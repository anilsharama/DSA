# DSA - Data Structures and Algorithms in Java

A collection of common data structures and algorithms implemented in Java for learning and reference purposes.

## Project Structure

```
src/
├── main/java/com/anilsharama/dsa/  # Implementation of data structures and algorithms
└── test/java/com/anilsharama/dsa/  # Unit tests
```

## Technologies Used

- Java 17
- Maven 3.x
- JUnit 5.10.2

## Building the Project

```bash
mvn clean compile
```

## Running Tests

```bash
mvn test
```

## Current Implementations

### Arrays
- `ArrayUtils`: Common array operations
  - `findMax()`: Find maximum element in an array
  - `findMin()`: Find minimum element in an array
  - `reverse()`: Reverse an array in place

## Requirements

- Java 17 or higher
- Maven 3.x
